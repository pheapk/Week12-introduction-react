Introduction to React

Please review the concepts covered this week.

React

React is a JavaScript library which allows you to quickly and efficiently create user interface (UI) elements with declarative language.

Components

A component is a building block of a React application. A React application typically has many components. Each component describes how a section of the UI (user interface) should look and function.

Here is a component that creates a header element with the words "Hello World!" inside it. Notice that it's created using JavaScript instead of HTML, even though it makes use of the HTML element. We can do this by using JSX, the JavaScript syntax extension that lets us use HTML in our JavaScript. 

const Greeting = () => <h1>Hello World!</h1>;

JSX

JSX is a JavaScript syntax extension that allows you to write HTML in JavaScript. By doing so, you can create React components much more quickly rather than writing them out entirely by hand in plain JavaScript. If you do not use JSX, you have to explicitly tell JavaScript to use the React library to create an element. JSX is used in place of vanilla JavaScript files in a React application for that reason.

Here is a code snippet of a React component created without JSX, from the React.js documentation: 

class Hello extends React.Component {
  render() {
    return React.createElement('div', null, `Hello ${this.props.toWhat}`);
  }
}

ReactDOM.render(
  React.createElement(Hello, {toWhat: 'World'}, null),
  document.getElementById('root')
);

And here is the same element created using JSX, from the React.js documentation: 

class Hello extends React.Component {
  render() {
    return <div>Hello {this.props.toWhat}</div>;
  }
}

ReactDOM.render(
  <Hello toWhat="World" />,
  document.getElementById('root')
);

Notice how using JSX makes the syntax much cleaner and easier to understand, as well as considerably shorter.

Destructuring

Destructuring is a convenient way of extracting multiple values from data stored in objects and arrays and storing them in variables.

Here is an example of an object being destructured: 

const obj = { first: 'Mark', last: 'Smith' };
const {firstName: first, lastName: last} = obj;
    // first = 'Mark'; last = 'Smith'

Here is a way to assign variables using destructuring:

let b
